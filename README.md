# 🪨📄✂️ Rock Paper Scissors Game (Python)

A simple command-line Rock–Paper–Scissors game built using Python. The player competes against the computer, which randomly selects its move.

---

## 📌 Features

- Classic Rock, Paper, Scissors gameplay
- ASCII art visuals for each choice
- Random computer selection
- Win / Lose / Draw logic
- Beginner-friendly Python code

---

## 🧠 Game Rules

- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock
- Same choice results in a Draw
